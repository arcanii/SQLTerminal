# SQLTerminal 1.1.4

## Editor
- **SQL syntax highlighting** — keywords, strings, comments, and numbers are now
  coloured as you type.

## History & snippets
- **Persistent, searchable query history** — the queries you run are remembered
  across launches (and shared across windows). Open the **History & Snippets**
  panel from the toolbar, search, and click any entry to load it back into the
  editor.
- **Snippets** — save the current query as a named, reusable snippet and insert
  it later from the same panel.

## Results
- **Sortable columns** — click a result header to sort by that column (click
  again to reverse). Sorting is numeric for number columns and natural otherwise.
- **Cell detail view** — right-click a cell → **View value…** to see the full
  value in a scrollable panel, with a **Format JSON** toggle for JSON values.
  Long cells now truncate to one line to keep the table tidy.

## Help
- A new **Help ▸ SQLTerminal Help** (**⌘?**) window — an offline reference of
  keyboard shortcuts, features, and where to find things.

## About
- The About window now uses the new transparent logo.
